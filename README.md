## 📌 Sobre o projeto

Scanner de segurança desenvolvido em Python para estudo de redes, programação e conceitos de cybersecurity.

A ferramenta realiza análise básica de portas TCP e gera relatórios dos resultados.

---

## 🚀 Funcionalidades

✅ Scanner de portas TCP  
✅ Identificação de IP  
✅ Detecção de portas abertas  
✅ Relatório automático  
✅ Interface via terminal
